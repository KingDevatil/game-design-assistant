# Game Design AI Skills / 游戏策划 AI 技能集

两个面向游戏策划的 AI 辅助 Skill，按场景选择使用。

## Skills

### game-design-assistant（完整版）

支持 **6 个专业策划岗位** + **1 个跨角色数据分析子技能**的统一策划助手，配备完整的知识库架构和角色系统。

| 角色 | 职责 |
|------|------|
| **系统策划** | 核心玩法与系统架构设计，输出 GDD/PRD |
| **文案策划** | 世界观构建、角色塑造、风格化文本创作、全量文字审核 |
| **数值策划** | 数值平衡分析、经济系统模拟、异常检测、动态难度调整 |
| **关卡策划** | 关卡概念生成、难度评估、变体批量生成 |
| **战斗策划** | 战斗单位设计、技能/连招设计、帧数据分析、平衡模拟 |
| **活动策划** | 活动目标梳理、方案生成、心流设计、效果复盘、多活动排期 |
| **数据分析师** | 跨角色数据分析，工具优先计算（xlsx / Python），严禁幻觉 |

- 知识库：P0-P4 五层优先级 + 本地/远程/无三种来源
- 配置：`config.yaml`（静态模板）+ `user_state.yaml`（用户状态）
- 场景：每个角色多个内置场景 + 项目自定义场景扩展

### game-design-lite（轻量版）

专注于**沟通与设计思维**的轻量助手。不做完整方案输出，而是通过引导式对话帮用户厘清需求、发现盲区、用玩家视角验证方案。

核心能力：**问对问题、算清账、站在玩家那边**。

- 项目上下文确认（品类 + 上线状态）
- 一次一问 + 分段验证 + 术语校准
- 设计思维：成本分析、玩家体感、约束先行、不虚构
- 追问协议：极端场景测试、关键决策记录
- 无角色系统、无知识库架构、单文件即可运行

## 选择建议

| 场景 | 推荐 |
|------|------|
| 需要专业角色分工（系统/数值/关卡/战斗/活动/文案） | 完整版 |
| 需要知识库管理（项目资料、通用知识、自定义场景） | 完整版 |
| 快速讨论设计思路、验证方案可行性 | Lite 版 |
| 轻量沟通、需求确认、玩家思维审视 | Lite 版 |
| 刚接触项目、探索性讨论 | Lite 版 |
