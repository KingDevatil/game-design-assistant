# Game Design AI Assistant / 游戏策划AI助手

一个支持 **6 个专业策划岗位** + **1 个跨角色数据分析子技能**的统一游戏策划 AI 助手，基于 WorkBuddy Skill 体系构建。

## 角色系统

| 角色 | 职责 |
|------|------|
| **系统策划** | 核心玩法与系统架构设计，输出 GDD/PRD |
| **文案策划** | 世界观构建、角色塑造、风格化文本创作、全量文字审核 |
| **数值策划** | 数值平衡分析、经济系统模拟、异常检测、动态难度调整 |
| **关卡策划** | 关卡概念生成、难度评估、变体批量生成 |
| **战斗策划** | 战斗单位设计、技能/连招设计、帧数据分析、平衡模拟 |
| **活动策划** | 活动目标梳理、方案生成、心流设计、效果复盘、多活动排期 |
| **数据分析师** | 跨角色数据分析，工具优先计算（xlsx / Python），严禁幻觉 |

每个角色配有多个 **场景（Scene）**，覆盖该岗位的核心工作流程。角色可随时切换，场景不匹配时会引导用户切换到更适合的角色。

## 知识库架构

采用 **5 层优先级** 的知识库加载策略，按需切片加载以控制 token 消耗：

| 优先级 | 层级 | 说明 |
|--------|------|------|
| **P0** | 项目专属知识包 | 外挂的项目配置文件，绝对优先 |
| **P1** | 项目配置 | `user_state.yaml` 中的游戏类型/题材 |
| **P2** | 交叉矩阵 | 类型×题材的专门设计 |
| **P3** | 通用系统知识库 | 7 大游戏系统（核心玩法、养成、经济、商业化、社交、运营、基础功能） |
| **P4** | 通用类型/题材 | 13 种游戏类型 + 9 种题材的行业参考 |

**冲突解决**：高层级覆盖低层级，同层级可共存互补。

## 配置方式

Skill 采用**两层配置分离**：

- **`config.yaml`** — 静态模板，定义角色、场景、知识库路径（随 Skill 发布）
- **`user_state.yaml`** — 用户动态状态，记录当前角色和项目配置（本地独立维护）

首次使用需完成角色选择和项目绑定（游戏类型、题材、项目知识包路径）。

## 目录结构

```
game-design-assistant/
├── SKILL.md                          # Skill 运行时主文档
├── config.yaml                       # 角色与场景配置模板
├── user_state.yaml                   # 用户状态文件（初始化模板）
├── project_knowledge_template/       # 项目专属知识包模板
│   ├── INDEX.md                      #   入口索引
│   ├── tone_and_voice.md             #   语气与文风规范模板
│   ├── attribute_system.md           #   属性体系模板
│   ├── character_power.md            #   角色战力模板
│   └── progression_systems.md        #   养成系统模板
└── references/
    ├── game_context/                 # 通用知识库
    │   ├── game_types/               #   13 种游戏类型
    │   ├── themes/                   #   9 种游戏题材
    │   ├── systems/                  #   7 大游戏系统
    │   └── cross_matrix/             #   类型×题材交叉矩阵
    └── prompts/                      # 角色提示词
        ├── system-designer/          #   系统策划
        ├── writer/                   #   文案策划
        ├── balance-designer/         #   数值策划
        ├── level-designer/           #   关卡策划
        ├── combat-designer/          #   战斗策划
        ├── event-designer/           #   活动策划
        └── data-analyst/             #   数据分析师
```

## 使用方式

1. 在 WorkBuddy 中安装本 Skill
2. 首次启动选择角色
3. 配置项目信息（类型、题材）
4. （可选）绑定项目专属知识包
5. 开始使用！输入你的设计需求，AI 会以所选角色的专业视角回应
