# 游戏类型：Gacha RPG / 数值卡牌（抽卡养成RPG）

> **对标参考**：《阴阳师》、《Fate/Grand Order》、《崩坏：星穹铁道》、《碧蓝航线》、《明日方舟》、《火影忍者》手游、《赛马娘》  
> **核心标签**：抽卡收集、角色养成、编队战斗、数值驱动、长周期运营、多系统耦合

---

## 核心循环

```
抽卡获取角色 → 养成强化（升级/升星/技能/装备） → 编队上阵 → 战斗推图/PVP → 获取资源 → 继续抽卡/养成
```

## 系统架构特征

| 系统模块 | 设计要点 |
|---|---|
| **抽卡系统（Gacha）** | 核心付费点，角色/装备按稀有度分级（N/R/SR/SSR/UR），保底机制（硬保底+软保底），UP池/限定池/常驻池 |
| **角色系统** | 大量角色（50-300+），每个角色有独特立绘、技能、语音、剧情，稀有度决定基础强度 |
| **养成系统** | 多维度深度养成：等级、升星/突破、技能等级、装备/圣遗物/御魂、天赋/命座/潜能、羁绊/好感 |
| **战斗系统** | 回合制、即时制、塔防、横版格斗等形式，支持自动/手动切换，编队多角色协同 |
| **编队系统** | 上阵3-6名角色，考虑职业/属性/阵营搭配，前中后排站位，队长技能 |
| **PVE内容** | 主线剧情、材料副本、日常副本、限时活动、高难挑战、无限爬塔/深渊 |
| **PVP内容** | 竞技场排名、段位赛、公会战、跨服战、异步PVP（防守阵容） |
| **经济系统** | 多货币体系（免费货币+付费货币+活动货币），体力系统限制进度，资源产出与消耗精确匹配 |
| **付费设计** | 角色/装备抽卡、月卡、战令/通行证、体力购买、养成礼包、皮肤外观，**抽卡是绝对核心付费** |

## 核心设计要素

### 抽卡机制设计
| 要素 | 说明 | 设计要点 |
|---|---|---|
| **稀有度分布** | N(40%) / R(30%) / SR(20%) / SSR(8%) / UR(2%) | 根据项目调整，高稀有度要有足够吸引力 |
| **保底机制** | 硬保底（90抽必出SSR）、软保底（70抽后概率递增） | 保底是法律要求和玩家心理预期 |
| **UP池** | 特定角色概率提升（占SSR的50%或100%） | UP角色设计要足够吸引人 |
| **天井/兑换** | 抽卡次数积累兑换指定角色（如300抽自选） | 给非酋兜底，给氪佬目标 |
| **免费获取** | 日常/活动产出抽卡券，保证免费玩家有稳定获取途径 | 月卡党每月可获得XX抽 |

### 角色养成深度
| 养成维度 | 说明 | 资源来源 | 深度 |
|---|---|---|---|
| **等级** | 基础属性成长 | 经验道具/副本 | 中 |
| **升星/突破** | 大幅提升属性，解锁新技能/上限 | 同名卡/通用碎片 | 深 |
| **技能等级** | 提升技能效果和倍率 | 技能书/金币 | 中 |
| **装备/圣遗物** | 随机词条装备，毕业难度高 | 副本/活动 | 极深 |
| **天赋/命座** | 重复抽取同名角色解锁 | 抽卡/通用道具 | 深 |
| **羁绊/好感** | 特定角色组合加成，独立剧情 | 日常互动/礼物 | 中 |

### 编队与战斗
| 要素 | 说明 | 设计要点 |
|---|---|---|
| **职业分工** | 坦克/输出/治疗/辅助/控制 | 明确分工，鼓励完整编队 |
| **属性克制** | 火→草→水→火，或光暗互克 | 克制倍率要显著但非绝对 |
| **阵营/种族加成** | 同阵营角色上阵触发加成 | 鼓励收集同阵营角色 |
| **前中后排** | 不同位置受击优先级不同 | 坦克前排、输出中后排 |
| **自动战斗** | 绝大多数战斗可自动完成 | AI释放技能逻辑要合理 |
| **战力系统** | 综合数值评分，卡关时明确提示 | 战力≠实际强度，但指导意义强 |

## 关键设计约束

1. **付费与免费平衡**：免费玩家要有完整体验，付费玩家加速进度，但不能让免费玩家完全无法参与高难内容
2. **抽卡公平感**：概率透明、保底明确、有免费获取途径，避免"骗氪"口碑
3. **养成深度控制**：养成线太多太深容易导致疲劳，需要设计减负机制（一键升级、批量强化）
4. **内容消耗**：新角色推出速度 vs 玩家养成速度要匹配，避免"抽到了养不起"
5. **社交设计**：公会、助战、PVP排名提供社交动力，但避免过度社交压力
6. **长草期管理**：活动期间肝，活动后闲，需设计常驻玩法填充长草期
7. **数值膨胀**：新角色强度要控制，避免"新卡必爆旧卡"导致玩家流失

## 常见策划产出物
- 角色设计文档（立绘需求、技能组、定位、背景故事）
- 抽卡概率与保底设计（概率模型、期望值、付费测算）
- 养成系统设计（消耗曲线、属性成长、解锁节点）
- 战斗数值配置（伤害公式、克制倍率、战力公式）
- 副本/关卡设计（难度曲线、敌人配置、掉落表）
- 活动方案（限定角色UP、养成加速、PVP赛季）
- 经济系统表（资源产出、消耗、通货膨胀控制）
