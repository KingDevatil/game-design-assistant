# 游戏类型：Simulation（模拟经营游戏）

> **对标参考**：《模拟城市》、《星露谷物语》、《动物园之星》、《双点医院》、《江南百景图》、《桃源深处有人家》  
> **核心标签**：资源管理、建设规划、时间经营、成就感、慢节奏、创造自由

---

## 核心循环

```
获取资源 → 建设/生产 → 满足需求 → 解锁新内容 → 扩大规模 → 优化效率
```

## 主要子类型

| 子类型 | 特征 | 代表 |
|---|---|---|
| **城市建设** | 规划城市布局，管理人口、交通、公共设施 | 《模拟城市》、《都市：天际线》 |
| **农场经营** | 种植、养殖、采集、季节变化 | 《星露谷物语》、《牧场物语》 |
| **商业模拟** | 经营商店/餐厅/医院/游乐园 | 《双点医院》、《过山车之星》 |
| **生活模拟** | 虚拟生活、社交、职业发展 | 《模拟人生》 |
| **沙盒创造** | 高度自由的建设和创造 | 《我的世界》（创造模式）、《动物森友会》 |

## 系统架构特征

| 系统模块 | 设计要点 |
|---|---|
| **资源系统** | 多种资源（金钱、材料、人口、食物、电力），资源间有生产和消耗关系 |
| **建设系统** | 放置建筑/设施，考虑布局效率（相邻加成、交通覆盖、美观度） |
| **生产链** | 原材料→加工→成品，形成产业链（如：小麦→面粉→面包） |
| **居民/NPC系统** | 居民有需求（住房/工作/娱乐/教育），满意度影响发展 |
| **时间系统** | 昼夜交替、季节变化、作物生长周期 |
| **任务/目标** | 短期目标引导玩家，长期目标提供方向 |
| **付费设计** | 买断制或F2P（加速道具、扩建、装饰皮肤、广告去除） |

## 核心设计要素

| 要素 | 说明 | 设计要点 |
|---|---|---|
| **布局策略** | 建筑位置影响效率和美观 | 道路连接、服务范围、相邻加成 |
| **供需平衡** | 生产与消耗要匹配，避免过剩或短缺 | 动态调整、仓储缓冲 |
| **等待机制** | 建设和生产需要时间 | 可用加速道具或广告跳过 |
| **收集成就** | 收集图鉴、成就系统、博物馆 | 满足收集癖和成就感 |
| **装饰自由** | 允许玩家自由装饰和美化 | 增加创造性和分享欲 |

## 关键设计约束

1. **节奏把控**：不能太肝（强迫定时上线），也不能太闲（没有目标感）
2. **复杂度阶梯**：前期简单，逐步引入新系统和机制，避免信息过载
3. **付费侵入性**：F2P模拟经营容易过度推销，影响体验
4. **长期目标**：需要有足够的长线内容支撑玩家持续游玩
5. **移动端适配**：操作简化，UI适配小屏幕，离线收益

## 常见策划产出物
- 建筑/设施设计表（成本、效果、占地、前置条件）
- 生产链设计（原料→产物→时间→利润）
- 居民需求设计（满意度公式、需求层级）
- 任务/成就系统设计（目标、奖励、解锁条件）
- 经济平衡表（产出、消耗、通货膨胀控制）
