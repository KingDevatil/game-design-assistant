# 题材：武侠 / 仙侠（Wuxia / Xianxia）

> **核心气质**：江湖情义、快意恩仇、修真问道、逆天改命、长生逍遥  
> **代表作品**：《射雕英雄传》、《仙剑奇侠传》、《诛仙》、《凡人修仙传》、《逆水寒》、《一念逍遥》

---

## 核心特征

### 武侠 vs 仙侠 区别
| 维度 | 武侠（Wuxia） | 仙侠（Xianxia） |
|---|---|---|
| **世界观** | 低魔，内力轻功为主，不脱离物理规律 | 高魔，飞天遁地、法宝神通、渡劫飞升 |
| **力量体系** | 内功、招式、兵器、轻功 | 灵力、法术、法宝、丹药、阵法 |
| **核心追求** | 侠义、武功天下第一、江湖地位 | 修仙长生、渡劫飞升、大道无极 |
| **代表作品** | 《射雕英雄传》、《笑傲江湖》 | 《诛仙》、《凡人修仙传》 |

### 世界观核心元素
| 元素 | 内容 | 游戏化运用 |
|---|---|---|
| **势力体系** | 正派（少林/武当/峨眉）、邪派（魔教/血刀门）、中立（丐帮/逍遥派） | 门派系统、阵营对抗 |
| **修炼体系** | 武侠：内力境界（三流→二流→一流→宗师→绝世）<br>仙侠：修仙境界（炼气→筑基→金丹→元婴→化神→渡劫→大乘→飞升） | 角色等级/境界系统 |
| **兵器谱** | 倚天剑、屠龙刀、玄铁重剑、飞剑、法宝 | 装备系统、专属武器 |
| **灵兽/坐骑** | 神雕、火麒麟、仙鹤、青龙 | 宠物/坐骑系统 |
| **秘籍/功法** | 九阳真经、乾坤大挪移、御剑术 | 技能书、功法系统 |
| **丹药/灵草** | 大还丹、九转金丹、万年灵芝 | 消耗品、炼药系统 |

## 文案风格指南

### 命名规范
- **角色名**：姓+名，常带江湖称号（如"北乔峰"、"西毒欧阳锋"）
- **技能名**：四字短语，带有诗意和力量感（如"降龙十八掌"、"万剑归宗"）
- **装备名**：材质+器型+品质修饰（如"玄铁重剑"、"九天玄女绫"）
- **地名**：山脉、洞天、古城（如"华山之巅"、"蜀山锁妖塔"、"洛阳城"）

### 描述风格
- **武功描述**：强调气势、意境、内力流转（如"掌风所至，龙吟虎啸，气浪翻涌"）
- **场景描述**：诗意与武侠感并存（如"月下竹林，剑光如练，一叶飘落于刃上分为两半"）
- **境界突破**：渡劫/突破时强调天地异象和内心感悟

## 系统设计影响
- **门派系统**：不同门派有独特武功和剧情线
- **功法/秘籍系统**：收集秘籍学习武功，可搭配组合
- **轻功/御剑系统**：移动方式体现题材特色
- **炼丹/锻造系统**：生活玩法，产出丹药和装备
- **江湖事件**：随机触发的江湖传闻、奇遇、悬赏任务
