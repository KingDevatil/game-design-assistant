# 题材：神话（Mythology）

> **核心气质**：神性与人性、命运与抗争、史诗传奇、神圣与亵渎  
> **代表作品**：《战神》系列、《哈迪斯》、《神界：原罪2》、《SMITE》、《江南百景图》（中国神话元素）

---

## 核心特征

### 主要神话体系
| 体系 | 特征 | 代表神祇 | 代表作品 |
|---|---|---|---|
| **希腊神话** | 神人同形同性，神有缺陷，命运不可违 | 宙斯、雅典娜、哈迪斯、波塞冬 | 《战神》、《哈迪斯》 |
| **北欧神话** | 宿命论、诸神黄昏、勇敢与荣耀 | 奥丁、索尔、洛基、芙蕾雅 | 《战神4/5》、《刺客信条：英灵殿》 |
| **埃及神话** | 死后审判、太阳崇拜、法老神性 | 拉、阿努比斯、荷鲁斯、奥西里斯 | 《刺客信条：起源》、《SMITE》 |
| **中国神话** | 天庭体系、修仙成神、法宝神通 | 玉皇大帝、孙悟空、哪吒、女娲 | 《黑神话：悟空》、《封神榜》 |
| **日本神道** | 万物有灵、神隐、妖怪文化 | 天照、八岐大蛇、稻荷神 | 《仁王》、《阴阳师》 |
| **克苏鲁神话** | 宇宙恐怖、不可名状、人类渺小 | 克苏鲁、奈亚拉托提普、犹格·索托斯 | 《血源诅咒》、《沉没之城》 |

### 世界观核心元素
| 元素 | 内容 | 游戏化运用 |
|---|---|---|
| **神祇体系** | 主神、从神、半神、英雄，神际关系复杂 | 角色池、势力系统 |
| **神器/圣物** | 神赐予的武器和道具，带有神性力量 | 装备系统、专属武器 |
| **神话生物** | 龙、凤凰、九头蛇、巨狼、狮鹫、妖怪 | 怪物设计、宠物/坐骑 |
| **圣地/神域** | 奥林匹斯山、阿斯加德、冥界、天庭 | 地图/副本设计 |
| **命运/预言** | 神谕、宿命、因果轮回 | 剧情驱动、任务设计 |

## 文案风格指南

### 命名规范
- **神名**：直接使用神话原名（如"宙斯 Zeus"、"奥丁 Odin"、"孙悟空"）
- **神器名**：神话原名或意译（如"雷神之锤 Mjolnir"、"如意金箍棒"）
- **地名**：神话中的地名（如"冥界 Hades"、"高天原"、"昆仑山"）
- **技能名**：引用神话典故（如"雷霆万钧"、"诸神黄昏"、"三昧真火"）

### 描述风格
- **神圣感**：描述神明时强调威严、光辉、不可侵犯（如"雷霆缠绕着奥林匹斯之王的权杖"）
- **史诗感**：战斗和事件描述要有宏大叙事感
- **人性刻画**：即使是神也要有人的情感和矛盾

## 系统设计影响
- **神祇选择/信仰系统**：玩家选择信仰某位神明，获得对应神力
- **神器收集**：收集神话中的著名神器，每件有独特能力
- **神话副本**：进入神话中的著名场景（冥界、天庭、瓦尔哈拉）
- **命运系统**：预言/命运影响剧情走向，玩家可抗争或顺应
- **变身/召唤系统**：借用神明力量变身或召唤神话生物助战
