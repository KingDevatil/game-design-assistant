# 题材：校园 / 日常（School Life / Slice of Life）

> **核心气质**：青春、成长、友情、恋爱、梦想、小确幸、淡淡的忧伤  
> **代表作品**：《女神异闻录》系列、《弹丸论破》、《心跳文学部》、《中国式家长》、《Summer Pockets》

---

## 核心特征

### 校园子类型
| 子类型 | 特征 | 代表 |
|---|---|---|
| **青春校园** | 日常生活、社团活动、考试、恋爱、友情 | 《女神异闻录》、《轻音少女》 |
| **悬疑校园** | 校园背景下的推理、悬疑、恐怖 | 《弹丸论破》、《心跳文学部》 |
| **超自然校园** | 校园+超能力/魔法/灵异元素 | 《魔法禁书目录》、《Charlotte》 |
| **恋爱模拟** | 以恋爱为核心，多角色攻略路线 | 《心跳回忆》、《夏日口袋》 |
| **养成模拟** | 培养角色成长，经历人生阶段 | 《中国式家长》、《偶像大师》 |

### 世界观核心元素
| 元素 | 内容 | 游戏化运用 |
|---|---|---|
| **时间结构** | 学期制、课程表、假期、毕业季 | 游戏时间推进系统 |
| **空间结构** | 教室、操场、图书馆、社团活动室、天台、便利店 | 场景/关卡设计 |
| **人际关系** | 同学、老师、朋友、恋人、 rivals、社团成员 | NPC系统、羁绊系统 |
| **成长主题** | 学业、梦想、自我认同、友情、初恋、离别 | 剧情和角色成长 |
| **日常活动** | 上课、考试、社团、打工、约会、购物 | 玩法系统 |

### 经典角色标签
| 类型 | 核心标签 | 代表风格 |
|---|---|---|
| **学霸** | 成绩优异、冷静、理性、偶尔天然 | 《女神异闻录》喜多川祐介 |
| **运动系** | 热血、直率、团队意识强 | 各种体育社团主力 |
| **文艺系** | 内向、敏感、有才华、爱读书/画画 | 文学社、美术社成员 |
| **辣妹/潮人** | 时尚、外向、消息灵通 | 班级中心人物 |
| **神秘转学生** | 来历不明、有秘密、引人注目 | 推动剧情的关键角色 |
| **学生会长** | 责任感强、认真、偶尔呆萌 | 权力与能力的象征 |

## 文案风格指南

### 命名规范
- **角色名**：常见日本/中国学生名，亲切自然（如"樱木花道"、"林星野"）
- **学校名**：普通学校名或带有特点的虚构名（如"私立希望之峰学园"）
- **社团名**：真实社团名（篮球部、轻音乐部、文学社、推理社）
- **物品名**：日常用品（便当中、练习册、情书、社团申请表）

### 描述风格
- **日常描述**：细腻、温暖、带有生活气息（如"夕阳把教室染成金色，粉笔灰在光柱中飘浮"）
- **对话风格**：自然口语化，带有青春特有的羞涩、冲动、真诚
- **情感表达**：含蓄而真挚，不直白说教，通过细节体现情感

## 系统设计影响
- **时间管理系统**：日程安排，选择做什么影响属性和关系
- **羁绊/好感系统**：与不同角色互动提升好感，解锁剧情
- **属性养成**：学习、运动、艺术、魅力等属性影响事件触发
- **多周目设计**：不同选择导致不同结局，鼓励重复游玩
- **迷你游戏**：考试、体育比赛、社团活动等玩法嵌入
