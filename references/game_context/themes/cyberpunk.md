# 题材：赛博朋克（Cyberpunk）

> **核心气质**：高科技低生活、企业 dystopia、霓虹美学、黑客反叛、身份认同  
> **代表作品**：《银翼杀手》、《攻壳机动队》、《赛博朋克2077》、《神经漫游者》、《 VA-11 Hall-A》

---

## 核心特征

### 世界观核心元素
| 元素 | 内容 | 游戏化运用 |
|---|---|---|
| **社会结构** | 巨型企业控制政府，贫富差距极端，底层生活在贫民窟 | 势力体系、阵营对抗 |
| **科技元素** | 义体改造、神经接口、虚拟现实、AI、无人机、全息投影 | 装备/技能系统 |
| **城市景观** | 高楼林立的富人区 vs 阴暗潮湿的贫民窟，霓虹灯、全息广告、雨水 | 地图/场景设计 |
| **核心冲突** | 个人自由 vs 企业控制、人类本质 vs 机械改造、信息自由 vs 监控 | 剧情主题 |
| **黑客文化** | 网络入侵、信息窃取、数字空间的"牛仔" | 黑客玩法系统 |

### 视觉风格
- **主色调**：霓虹紫、电光蓝、荧光粉、暗绿，搭配深黑背景
- **UI元素**：故障艺术（Glitch）、终端代码、全息投影、数据流
- **场景特征**：雨天、霓虹反射、拥挤的街道、巨型广告屏、地下俱乐部

## 文案风格指南

### 命名规范
- **人名**：混合东西方（如"草薙素子"、"V"、"强尼·银手"）
- **企业名**：抽象+力量感（如"荒坂公司"、"康陶"、"军用科技"）
- **地区名**：日文/中文音译+英文混合（如"夜之城"、"新埼玉"、"第9区"）
- **技术名词**：缩写+型号（如"斯安威斯坦"、"歧路司义眼"、"单分子线"）

### 描述风格
- **场景描述**：雨夜霓虹、机械与肉体的边界、信息过载的感官体验
- **对话风格**：街头俚语、黑客术语、带有反叛和愤世嫉俗的语气
- **叙事基调**：黑暗、 gritty、偶尔带有黑色幽默

## 系统设计影响
- **义体改造系统**：身体部位模块化改造，不同义体带来不同能力
- **黑客系统**：网络入侵、控制电子设备、窃取数据、精神攻击
- **信息收集**：通过侦查、窃听、黑入获取任务情报
- **势力声望**：不同企业和帮派有独立声望，影响商店、任务、剧情
- **犯罪系统**：警察响应等级、通缉、腐败警察可贿赂
