# Scene: EVT-ui - 活动UI/UX设计

## 场景标识
- **ID**: EVT-ui
- **名称**: 活动UI/UX设计
- **岗位**: 活动策划
- **优先级**: P1（Phase 2）
- **适用场景**: 当你需要设计活动的界面布局、交互流程和视觉规范建议时。

## 场景目的
输入活动方案，输出活动的界面布局方案、交互流程、视觉规范建议，确保活动界面既美观又易用。

## 输入模板

```markdown
## 活动UI设计输入

### 活动信息
- 活动名称：_______
- 活动主题/风格：_______（如：春节喜庆、暗黑神秘）
- 主色调：_______

### 界面需求
- 需要设计的界面：_______
- 核心操作：_______
- 必须展示的信息：_______

### 约束
- 屏幕尺寸：_______
- UI风格规范：_______
- 性能限制：_______
```

## 处理逻辑

1. **信息梳理**
   - 列出每个界面需要展示的所有信息
   - 按优先级分层

2. **布局设计**
   - 设计每个界面的布局（用文字描述+ASCII示意图）
   - 确保核心信息突出、操作便捷

3. **交互流程**
   - 设计玩家在活动中的操作流程
   - 标注每个操作的反馈

4. **视觉规范**
   - 配色、字体、图标、动效建议
   - 确保与活动主题和项目整体风格一致
