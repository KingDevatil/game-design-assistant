# Scene: CBT-sim - 对战平衡模拟

## 场景标识
- **ID**: CBT-sim
- **名称**: 对战平衡模拟
- **岗位**: 战斗策划
- **优先级**: P0（Phase 2）

## 场景目的
输入两个或多个战斗单位的属性和技能配置，模拟大量对战，分析胜率分布、克制关系、强度梯队，输出平衡性调优建议。

```markdown
### 平衡目标
- 理想胜率区间：_______%
- 可接受的最高胜率：_______%
- 是否存在必须存在的克制关系：_______
```

## 处理逻辑

1. **战力评估**
   - 计算每个单位的理论DPS、生存能力、机动性评分
   - 计算综合战力值

2. **对战模拟**
   - 基于属性和技能逻辑模拟对战过程
   - 考虑操作水平差异（引入随机因子）
   - 记录胜负、剩余HP、战斗时长

3. **胜率分析**
   - 生成对阵胜率矩阵
   - 计算每个单位的平均胜率
   - 识别过强/过弱单位

4. **克制关系分析**
   - 识别明显的克制链
   - 评估克制强度是否合理

5. **调优建议**
   - 对失衡单位给出属性/技能调整建议
