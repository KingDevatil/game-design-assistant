# 题材：奇幻（Fantasy）

## 核心特征
- 魔法是核心力量体系
- 存在非人类种族（精灵、矮人、兽人、龙等）
- 世界观通常有完整的历史、地理、神话体系
- 英雄旅程是经典叙事结构

## 常见子类型
| 子类型 | 特征 | 代表作品 |
|---|---|---|
| 高奇幻（High Fantasy） | 完整架空世界、史诗级冲突、善恶二元对立 | 《魔戒》、《魔兽世界》 |
| 黑暗奇幻（Dark Fantasy） | 残酷世界观、道德灰色地带、恐怖元素 | 《黑暗之魂》、《巫师》 |
| 剑与魔法（Sword & Sorcery） | 个人冒险为主、快节奏、重动作轻政治 | 《蛮王柯南》 |
| 都市奇幻（Urban Fantasy） | 魔法存在于现代都市 | 《哈利波特》（后期） |

## 文案风格指南
### 命名规范
- 地名：使用带有异域感的音节组合（如"艾泽拉斯"、"诺森德"）
- 人名：根据种族特征区分（精灵名优雅、矮人名粗犷）
- 物品名：使用形容词+名词结构，暗示稀有度和力量感

### 描述风格
- 物品描述：简短有力，暗示背景故事（如"这把剑曾属于最后一位龙骑士"）
- 技能描述：强调视觉效果和力量感
- 任务描述：突出英雄感和使命感

## 系统设计影响
- UI设计可融入魔法符文、古老卷轴等视觉元素
- 社交系统可设计种族阵营对立（如联盟vs部落）
- 经济系统可设计魔法材料采集和炼金系统
