# 题材：三国（Three Kingdoms）

> **历史背景**：东汉末年（184年黄巾起义）→ 群雄割据 → 魏蜀吴三国鼎立（220-280年）  
> **核心气质**：英雄主义、忠义、权谋、乱世豪情  
> **代表作品**：《三国演义》（罗贯中）、《真·三国无双》系列、《三国志》系列、《三国战纪》系列

---

## 文案风格指南

### 命名规范
- **角色名**：使用历史原名（关羽、赵云），可附加称号（武圣·关羽、常山赵子龙）
- **技能名**：引用典故或诗句（如"过五关斩六将"、"草船借箭"）
- **装备名**：历史兵器名+品质修饰（如"真·青龙偃月刀"、"传国玉玺"）
- **副本名**：经典战役名（如"赤壁之战·火攻篇"、"长坂坡·单骑救主"）

### 描述风格
- **武器描述**：强调历史典故和主人传奇（如"此刀随关羽斩颜良诛文丑，过五关斩六将，刀身留有百战痕迹"）
- **技能描述**：结合历史事件或人物特征（如"赵云七进七出长坂坡的无双枪法"）
- **角色小传**：突出人物核心矛盾和悲剧性（如吕布的"反复无常"与"天下无双"的冲突）

### 台词风格
| 角色类型 | 风格特征 | 示例 |
|---|---|---|
| 猛将 | 豪迈、简短、带有战场气息 | "某家在此，谁敢一战！" |
| 谋士 | 从容、深远、带有策略感 | "主公勿忧，亮有一计。" |
| 君主 | 霸气或仁厚，体现领导力 | "宁教我负天下人，休教天下人负我。" |
| 女性角色 | 柔美中见坚韧，或机智过人 | "妾身愿为君分忧。" |

### 禁止使用
- 现代网络用语、英文混合
- 与三国历史严重冲突的设定（如诸葛亮使用火器）
- 过度美化反面人物到脱离历史认知（如董卓成为正面角色需特别说明AU设定）

## 视觉风格指引

### 主色调建议
- **魏**：深蓝 + 黑色（冷峻、威严、肃杀）
- **蜀**：暗红 + 玄色（忠义、血性、沉稳）
- **吴**：深绿 + 青铜色（江东、水战、谋略）
- **群雄**：暗紫 + 灰色（乱世、割据、神秘）

### UI风格元素
- 卷轴、竹简、兵符、虎符等古代文书元素
- 水墨/工笔画风格的角色立绘
- 战旗、铠甲、兵器等装饰性边框
- 古风书法字体用于标题

## 系统设计影响

### 势力系统
- 角色按魏/蜀/吴/群雄划分阵营
- 同阵营角色触发羁绊加成
- 势力对抗玩法（如魏蜀吴三国争霸活动）

### 武器系统
- 每个知名武将设计专属武器
- 武器与角色绑定，强化角色特性
- 可设计武器觉醒/进阶系统（如青龙偃月刀→真·青龙偃月刀→武圣之刃）

### 兵法系统
- 将三十六计等兵法设计为策略卡牌/被动技能
- 战斗中可释放兵法改变战局（如火攻造成持续伤害）

### 剧情叙事
- 主线按三国演义经典章节推进
- 可设计IF线（如"假如关羽没有失荆州"）作为活动剧情
- 角色个人传记副本，深入挖掘人物故事
