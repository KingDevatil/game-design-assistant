# 题材：科幻（Science Fiction / Sci-Fi）

> **核心气质**：未来感、科技理性、人类命运、宇宙探索、未知恐惧  
> **代表作品**：《星际穿越》、《三体》、《质量效应》、《赛博朋克2077》（科幻子类型）、《EVE Online》

---

## 核心特征

### 世界观核心元素
| 元素 | 内容 | 游戏化运用 |
|---|---|---|
| **时间设定** | 近未来（50年内）→ 远未来（数百年至数千年）→ 星际时代 | 科技等级决定玩法 |
| **科技体系** | 太空航行、人工智能、基因改造、虚拟现实、量子技术、纳米科技 | 技能系统、装备系统 |
| **空间设定** | 地球、太空站、星际舰队、异星殖民地、虚拟现实空间 | 关卡/地图设计 |
| **冲突主题** | 人类 vs AI、文明冲突、资源争夺、生存危机、伦理困境 | 剧情和玩法驱动 |
| **势力体系** | 地球联邦、星际帝国、企业财团、反抗军、外星文明 | 阵营系统 |

### 科幻子类型
| 子类型 | 特征 | 代表作品 |
|---|---|---|
| **硬科幻** | 严格遵守物理定律，科技有理论基础 | 《三体》、《星际穿越》 |
| **太空歌剧** | 宏大叙事、星际战争、英雄传奇 | 《星球大战》、《质量效应》 |
| **后人类** | 基因改造、意识上传、人机融合 | 《攻壳机动队》、《银翼杀手》 |
| **末世科幻** | 科技灾难后的世界 | 《辐射》、《最后生还者》（偏废土） |
| **企业 dystopia** | 巨型企业控制世界 | 《赛博朋克2077》、《异形》 |

## 文案风格指南

### 命名规范
- **人名**：简洁、国际化、带有未来感（如"谢顿"、"安德森"、"V"）
- **地名**：编号+功能（如"殖民站Alpha-7"、"新东京"、"火星奥林匹斯城"）
- **技术名词**：缩写+数字（如"FTL引擎"、"量子通讯Q-Comm"、"神经接口V3"）
- **组织名**：功能性命名（如"星际联盟"、"泰坦军工"、"黑曜石集团"）

### 描述风格
- **科技描述**：强调参数、效率、突破性（如"零点能反应堆，能量转化率99.7%"）
- **场景描述**：宏大与冰冷并存（如"从舷窗望去，气态巨行星的风暴眼如同一只永恒凝视的眼睛"）
- **对话风格**：理性、简洁、带有军事或学术色彩

## 系统设计影响
- **UI设计**：全息投影、终端界面、数据流、极简几何风格
- **装备系统**：模块化科技装备，可升级/改装（武器配件、装甲模块）
- **技能系统**：科技能力（黑客入侵、无人机控制、能量护盾、瞬移装置）
- **经济系统**：星际贸易、资源开采、信用点货币
- **载具系统**：飞船、机甲、悬浮载具，可自定义和升级
