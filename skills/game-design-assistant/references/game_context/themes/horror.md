# 题材：恐怖（Horror）

> **核心气质**：恐惧、未知、压抑、心理扭曲、生存本能  
> **代表作品**：《生化危机》、《寂静岭》、《逃生》、《恐鬼症》、《第五人格》、《甜蜜之家》

---

## 核心特征

### 恐怖子类型
| 子类型 | 特征 | 代表 |
|---|---|---|
| **生存恐怖** | 资源稀缺、无力感、逃避多于战斗 | 《生化危机》、《逃生》 |
| **心理恐怖** | 氛围压抑、精神错乱、现实与幻觉模糊 | 《寂静岭》、《层层恐惧》 |
| ** jump scare** | 突然惊吓、快节奏、直播效果好 | 《玩具熊的五夜后宫》 |
| **合作恐怖** | 多人合作对抗恐怖元素 | 《恐鬼症》、《黎明杀机》（非对称） |
| **非对称对抗** | 1个杀手 vs 多个幸存者 | 《黎明杀机》、《第五人格》 |
| **克苏鲁恐怖** | 宇宙恐怖、不可名状、理智崩溃 | 《血源诅咒》、《沉没之城》 |

### 世界观核心元素
| 元素 | 内容 | 游戏化运用 |
|---|---|---|
| **恐怖来源** | 怪物、鬼魂、疯子、未知生物、超自然现象、人类自身 | 敌人/事件设计 |
| **场景设定** | 废弃医院、古堡、阴暗森林、精神病院、深海、太空站 | 地图设计 |
| **资源稀缺** | 弹药、医疗包、光源（电池/火柴）极其有限 | 生存机制 |
| **脆弱性** | 主角脆弱，无法正面对抗，只能逃跑或隐藏 | 战斗/躲避机制 |
| **理智系统** | 目睹恐怖场景降低理智，导致幻觉或失控 | 特殊机制 |

## 文案风格指南

### 命名规范
- **怪物名**：抽象、带有不祥感（如"三角头"、"恶灵"、"修女"、"瘦长鬼影"）
- **地点名**：普通地名+阴暗修饰（如"寂静岭"、"黑水精神病院"、"橡树巷14号"）
- **物品名**：旧、破损、血迹（如"生锈的钥匙"、"沾血的日记"）

### 描述风格
- **氛围营造**：强调听觉（脚步声、低语、心跳）、视觉（阴影、闪烁的灯光）
- **叙事方式**：碎片化叙事（日记、录音、环境细节），不直接告诉玩家真相
- **心理描写**：主角的恐惧、猜疑、逐渐崩溃的心理状态

## 系统设计影响
- **光照系统**：光源稀缺，黑暗区域有危险，手电筒/火柴有持续时间
- **声音设计**：脚步声、呼吸声、环境音是核心体验，声音会吸引敌人
- **躲藏机制**：柜子、床底、阴影中躲藏，屏住呼吸（QTE）
- **追逐战**：被追逐时的紧张感，狭窄通道、障碍物、死胡同
- **理智/压力系统**：恐怖事件积累压力，导致视野扭曲、操作异常
- **多人非对称**：杀手视角（强大但有弱点）vs 幸存者视角（脆弱但可合作）
