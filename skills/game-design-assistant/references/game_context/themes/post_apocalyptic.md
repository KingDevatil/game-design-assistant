# 题材：末世 / 废土（Post-Apocalyptic / Wasteland）

> **核心气质**：生存挣扎、人性考验、废墟美学、资源稀缺、重建希望  
> **代表作品**：《辐射》系列、《最后生还者》、《疯狂的麦克斯》、《地铁》系列、《明日之后》、《黎明觉醒》

---

## 核心特征

### 世界观核心元素
| 元素 | 内容 | 游戏化运用 |
|---|---|---|
| **灾难起因** | 核战争、丧尸病毒、气候灾难、陨石撞击、AI叛乱 | 世界观背景和敌人设计 |
| **世界状态** | 文明崩溃、城市废墟、辐射区、变异生物、资源枯竭 | 地图和关卡设计 |
| **生存要素** | 饥饿、口渴、疾病、辐射、体温、疲劳 | 生存玩法机制 |
| **势力体系** | 幸存者聚落、掠夺者帮派、军方残余、变种人、科学家团体 | 阵营和NPC势力 |
| **核心冲突** | 生存 vs 道德、个人 vs 群体、希望 vs 绝望 | 剧情选择和玩法驱动 |

### 废土子类型
| 子类型 | 特征 | 代表 |
|---|---|---|
| **核废土** | 核战后辐射世界，50年代复古未来主义 | 《辐射》系列 |
| **丧尸末世** | 丧尸病毒爆发，人类濒临灭绝 | 《最后生还者》、《求生之路》 |
| **气候末日** | 极端气候导致文明崩溃 | 《疯狂的麦克斯》、《地铁》 |
| **赛博末日** | 科技失控后的 dystopia | 《尼尔：机械纪元》 |

## 文案风格指南

### 命名规范
- **人名**：粗犷、简短、带有生存气息（如"马克斯"、"乔尔"、"独眼"）
- **地名**：旧地名+现状描述（如"锈蚀镇"、"死水城"、"旧地铁站"）
- **势力名**：功能化+威慑感（如"钢铁兄弟会"、"掠夺者联盟"、"清泉镇民兵团"）
- **物品名**：旧世界名称+状态（如"生锈的步枪"、"发霉的罐头"、"战前货币"）

### 描述风格
- **场景描述**：废墟、荒芜、压抑中偶有生机（如"倒塌的摩天大楼间，一株野花在裂缝中绽放"）
- **对话风格**：简洁、直接、带有戒备和不信任
- **叙事基调**：灰暗但不绝望，强调人性的坚韧和选择

## 系统设计影响
- **生存系统**：饥饿、口渴、体温、疾病、辐射等多维生存指标
- **资源稀缺**：弹药、医疗包、食物珍贵，鼓励探索和拾荒
- **制造系统**：用废料制作工具和武器（如"用钢管+胶带制作近战武器"）
- **基地/避难所建设**：安全屋升级、防御工事、资源种植
- **道德选择**：救或不救、分享或独占，影响剧情走向和NPC态度
- **天气/环境危害**：辐射风暴、酸雨、极寒/极热
