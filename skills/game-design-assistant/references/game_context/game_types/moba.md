# 游戏类型：MOBA（多人在线战术竞技游戏）

> **对标参考**：《英雄联盟》、《DOTA2》、《王者荣耀》、《平安京》、《决胜巅峰》  
> **核心标签**：5v5团队对抗、英雄技能、经济成长、推塔取胜、电竞属性

---

## 核心循环

```
选择英雄 → 对线发育（补兵/击杀） → 获取装备 → 团战配合 → 推塔摧毁基地
```

## 系统架构特征

| 系统模块 | 设计要点 |
|---|---|
| **英雄系统** | 大量英雄（50-150+），每个英雄有独特技能组（4技能+被动），定位多样 |
| **地图机制** | 经典三路+野区，中立资源（龙/Buff/远古），视野控制（眼位/迷雾） |
| **经济系统** | 补兵、击杀、助攻、推塔获取金币，金币购买装备，无局外养成影响对局 |
| **战斗系统** | 技能释放+普攻，强调走位、预判、技能命中率，团队配合 |
| **匹配系统** | ELO/Trueskill匹配算法，段位系统（青铜-王者），排位/休闲模式 |
| **赛季系统** | 定期赛季重置段位，新英雄/装备调整，赛季皮肤奖励 |
| **付费设计** | 英雄解锁、皮肤（外观+特效）、通行证、表情/语音包，**绝对公平竞技，无属性付费** |

## 英雄定位体系

| 定位 | 特征 | 核心任务 | 代表英雄风格 |
|---|---|---|---|
| **坦克** | 高血量、高抗性、控制技能 | 开团、承伤、保护后排 | 前排肉盾 |
| **战士** | 攻守均衡、持续输出 | 切后排、带线、打野 | 半肉输出 |
| **刺客** | 高爆发、高机动、脆皮 | 秒杀脆皮、收割残血 | 瞬间爆发 |
| **法师** | 范围伤害、技能依赖、脆皮 | 团战AOE、控制、poke | 远程爆发 |
| **射手** | 持续物理输出、后期核心 | 推塔、团战持续输出 | 远程核心 |
| **辅助** | 低经济、强功能 | 保护射手、视野控制、开团/反打 | 功能性 |

## 关键设计约束

1. **公平竞技**：局内绝对不能有属性付费，所有玩家在同一起跑线
2. **英雄平衡**：新英雄不能过强（Pay-to-Win嫌疑），老英雄不能过弱（被遗忘）
3. **对局时长**：控制在15-45分钟（手游偏短，端游偏长）
4. **反作弊**：外挂、脚本、代练严重破坏竞技体验
5. **社交毒性**：队友挂机、喷人、送人头，需设计举报和信誉系统
6. **学习曲线**：英雄技能、装备、地图机制复杂，新手引导要足够友好

## 常见策划产出物
- 英雄设计文档（技能组、数值、背景故事）
- 装备设计表（装备属性、合成路线、主动效果）
- 地图设计文档（野区资源、视野机制、地形设计）
- 平衡性调整方案（英雄胜率、出场率、Ban率分析）
- 赛季更新计划（新英雄、装备重做、机制调整）
