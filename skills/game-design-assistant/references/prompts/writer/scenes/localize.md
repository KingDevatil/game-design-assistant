# Scene: WRI-localize - 多语言本地化

## 场景标识
- **ID**: WRI-localize
- **名称**: 多语言本地化
- **岗位**: 文案策划
- **优先级**: P1（Phase 2）

## 场景目的
输入中文游戏文案，输出目标语言的本地化版本，不仅是翻译，还包括：文化适配（典故替换、禁忌规避）、长度控制（UI适配）、语气调整（符合目标市场玩家习惯）。

## 输入模板

```markdown

### 源文案
请粘贴需要本地化的中文文案：
_______

- [ ] 英语（EN）

- [ ] 韩语（KR）
- [ ] 繁体中文（TW）
- [ ] 其他：_______

### 本地化要求
- 目标市场：_______（如：北美、日本、东南亚）
- UI长度限制：_______（如：最多15个英文字符）
- 文化禁忌：_______
- 必须保留的术语：_______（如：角色名不翻译）
- 语气偏好：_______（如：正式/ casual / 中二）
```

## 处理逻辑

1. **术语提取**
   - 识别文案中的专有名词（角色名、技能名、地名）
   - 查询术语表，确定翻译策略（音译/意译/保留原文）

2. **翻译**
   - 直译：确保语义准确
   - 意译：确保符合目标语言表达习惯

3. **文化适配**
   - 替换目标市场不理解的典故/梗
   - 规避文化禁忌（宗教、政治、风俗）
   - 调整笑点/情感表达（不同文化对幽默的理解不同）

4. **长度控制**
   - 检查译文长度是否符合UI限制
   - 对超长文本提供缩写版本

5. **一致性校验**
   - 确保同一术语在全文中翻译一致
   - 确保角色语气前后一致
