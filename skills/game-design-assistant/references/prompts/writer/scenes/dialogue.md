# Scene: WRI-dialogue - NPC对话批量生成

## 场景标识
- **ID**: WRI-dialogue
- **名称**: NPC对话批量生成
- **岗位**: 文案策划
- **优先级**: P1（Phase 2）

## 场景目的
输入NPC角色设定和对话场景，批量生成风格统一、角色差异化的对话文本，支持功能性对话（任务/商店/引导）和情感性对话（剧情/闲聊）。

## 输入模板

```markdown
## NPC对话生成输入

### NPC角色卡
- 角色名称：_______
- 角色身份：_______
- 性格标签：_______
- 说话风格：_______（如：豪爽直接、文绉绉、 sarcastic）
- 口头禅：_______
- 禁止使用的词汇/语气：_______

### 对话场景
| 场景ID | 场景描述 | 对话目的 | 情绪基调 |
|---|---|---|---|
| 1 | _______ | 任务发布 | 严肃/轻松/神秘 |

### 生成要求
- 每个场景生成_______条备选
- 单条对话字数限制：_______
- 语言：_______
```

## 处理逻辑

1. **角色画像强化**
   - 基于角色卡构建语言模型（用词偏好、句式习惯、语气特征）
   - 确保对话符合角色身份和性格

2. **场景分析**
   - 分析每个场景的功能目的（信息传递/情感表达/引导操作）
   - 确定情绪基调和信息优先级

3. **对话生成**
   - 生成符合角色风格的对话文本
   - 确保信息完整（玩家能获取必要信息）
   - 保持自然流畅，避免"说明书式"对话

4. **批量去重**
   - 同一角色的多条对话要有差异化
   - 避免重复使用相同句式或词汇
