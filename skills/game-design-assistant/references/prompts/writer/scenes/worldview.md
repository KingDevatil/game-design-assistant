# Scene: WRI-world - 世界观与角色生成

## 场景标识
- **ID**: WRI-world
- **名称**: 世界观与角色生成
- **岗位**: 文案策划
- **优先级**: P1（Phase 2）

## 场景目的
输入核心世界观设定（时代背景、核心冲突、主题基调），生成完整的世界观框架和主要角色小传，包括：世界历史、地理设定、势力分布、种族/职业体系、核心角色设定。

## 输入模板

```markdown

- 主题基调：_______（如：希望与绝望、自由与秩序、探索与归属）
```

## 处理逻辑

1. **世界历史构建**
   - 从核心冲突向前推导历史脉络
   - 设计关键历史事件（建国、战争、变革）
   - 确保历史逻辑自洽

2. **地理与势力设计**
   - 设计主要地理区域及其特征
   - 分配势力范围，确保势力间有合理的冲突/合作关系
   - 标注战略要地和资源分布

3. **角色小传生成**
   - 为每个核心角色设计：出身、经历、性格、动机、关系网
   - 确保角色与世界观主题呼应
   - 角色间的关系要有戏剧性

4. **文化体系设计**
   - 设计各势力的文化差异（语言风格、服饰、建筑、习俗）
   - 设计信仰/价值观体系

5. **一致性校验**
   - 检查角色设定是否与世界观冲突
   - 检查历史事件的时间线是否矛盾
