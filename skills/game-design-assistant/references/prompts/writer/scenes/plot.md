# Scene: WRI-plot - 剧情逻辑校验

## 场景标识
- **ID**: WRI-plot
- **名称**: 剧情逻辑校验
- **岗位**: 文案策划
- **优先级**: P0（Phase 2）

## 场景目的
输入剧情文档或对话树，自动检测剧情中的逻辑矛盾、时间线错误、设定冲突、角色行为OOC（Out Of Character）等问题。

## 输入模板

```markdown

### 待校验内容
请粘贴剧情文本/对话树/世界观设定：
_______

### 校验范围
- [ ] 时间线一致性
- [ ] 角色设定一致性

- [ ] 世界观设定冲突
- [ ] 对话OOC检测
- [ ] 地理位置合理性

### 参考设定（用于对比校验）
- 角色设定：_______
- 世界观设定：_______
- 历史时间线：_______
```

## 处理逻辑

1. **时间线梳理**
   - 提取剧情中所有时间标记
   - 构建事件顺序图
   - 检测因果倒置或时间跳跃矛盾

2. **角色设定校验**
   - 提取角色在剧情中的行为和台词
   - 与角色设定卡对比，检测OOC
   - 检测角色知识/能力是否超纲（知道不该知道的事）

3. **因果关系校验**
   - 检查剧情转折是否有足够铺垫
   - 检查后果是否与原因匹配

4. **世界观一致性**
   - 检查剧情中使用的设定是否与世界观冲突
   - 检查地理/历史/文化细节是否一致

5. **输出问题清单**
   - 按严重程度分级
   - 给出修改建议
