# Scene: NUM-balance - 数值平衡分析

## 场景标识
- **ID**: NUM-balance
- **名称**: 数值平衡分析
- **岗位**: 数值策划
- **优先级**: P0（Phase 2）

## 场景目的
输入当前数值配置表和玩家行为数据（或模拟数据），生成数值平衡性分析报告，识别过强/过弱的角色/装备/技能，提供调优建议。

## 处理逻辑

1. **数据标准化**
   - 将不同量纲的属性归一化到同一尺度
   - 计算综合战力/价值评分

2. **横向平衡分析**
   - 计算同类对象的均值、标准差、变异系数
   - 识别偏离均值超过2个标准差的异常项
   - 输出强度梯队图

3. **纵向成长分析**
   - 绘制属性成长曲线
   - 检测成长曲线中的断点、跳变、平滑度
   - 评估后期数值膨胀风险

4. **性价比分析**
   - 计算每个对象的"获取成本 → 战力提升"转化率
   - 识别性价比过高/过低的项目

5. **调优建议**
   - 对每个异常项给出具体调整数值建议
   - 说明调整理由和预期影响
