# Scene: EVT-research - 同类型活动参考与调研分析

## 场景标识
- **ID**: EVT-research
- **名称**: 同类型活动参考与调研分析
- **岗位**: 活动策划
- **优先级**: P0（Phase 2）
- **适用场景**: 当你需要调研竞品同类活动，了解其设计亮点、奖励结构、节奏安排和玩家反馈时。

## 场景目的
输入活动类型和目标，收集竞品同类型活动的案例，分析其设计亮点、奖励结构、节奏安排、玩家反馈，输出调研报告。

## 输入模板

```markdown
## 活动调研输入

### 活动类型
- 活动大类：_______（如：节日活动/赛季活动/联动活动/版本活动）
- 具体形式：_______（如：签到/累充/排行榜/副本挑战/收集兑换）
- 目标用户：_______

### 竞品范围
- 直接竞品：_______
- 间接竞品：_______
- 参考案例：_______

### 调研重点
- [ ] 活动规则设计
- [ ] 奖励结构设计
- [ ] 节奏与阶段划分
- [ ] 付费点设计
- [ ] 玩家反馈与评价
```

## 处理逻辑

1. **竞品活动收集**
   - 列出竞品同类型活动的案例
   - 记录活动的基本信息（时间、规则、奖励）

2. **规则拆解**
   - 拆解每个竞品活动的核心规则
   - 对比规则复杂度、参与门槛、理解成本

3. **奖励结构分析**
   - 分析奖励类型、数量、获取难度
   - 评估奖励的吸引力和性价比

4. **节奏分析**
   - 分析活动的阶段划分和时间安排
   - 识别节奏设计的亮点

5. **玩家反馈收集**
   - 整理玩家对竞品活动的评价
   - 识别好评点和吐槽点

6. **可借鉴方案**
   - 总结可以借鉴的设计点
   - 标注需要规避的坑
