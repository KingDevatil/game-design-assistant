# Scene: EVT-copy - 活动文案生成

## 场景标识
- **ID**: EVT-copy
- **名称**: 活动文案生成
- **岗位**: 活动策划/文案策划
- **优先级**: P1（Phase 2）
- **适用场景**: 当你需要批量生成活动公告、规则说明、界面文案、邮件文案等全部文案时。

## 场景目的
输入活动方案，批量生成活动相关的全部文案：活动公告、规则说明、界面文案、邮件文案、推送文案，确保风格统一、信息准确、吸引力强。

## 输入模板

```markdown
## 活动文案生成输入

### 活动信息
- 活动名称：_______
- 活动主题：_______
- 核心奖励：_______
- 活动时间：_______
- 目标玩家：_______

### 文案需求
- [ ] 活动公告（全服）
- [ ] 活动规则说明
- [ ] 界面文案（按钮/标题/提示）
- [ ] 邮件文案（邀请/提醒/奖励）
- [ ] 推送文案（APP推送）
- [ ] 社交媒体文案（微信/微博）

### 风格要求
- 语气：_______（正式/活泼/热血/温馨）
- 长度限制：_______
- 必须包含的关键词：_______
- 禁止使用的词汇：_______
```

## 处理逻辑

1. **核心信息提取**
   - 从活动方案提取玩家最关心的信息：奖励、时间、玩法、规则
   - 确定每条文案的核心卖点

2. **文案分层设计**
   - 公告文案：全面、正式、信息完整
   - 推送文案：简短、吸引点击、突出奖励
   - 界面文案：简洁、清晰、引导性强
   - 邮件文案：亲切、有温度、行动召唤

3. **风格统一**
   - 确保所有文案的语气、用词风格一致
   - 符合项目整体Tone & Voice

4. **卖点突出**
   - 每条文案都有明确的吸引点
   - 奖励信息前置，降低理解成本
