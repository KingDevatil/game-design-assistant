# Scene: EVT-goal - 活动设计目的明确与梳理

## 场景标识
- **ID**: EVT-goal
- **名称**: 活动设计目的明确与梳理
- **岗位**: 活动策划
- **优先级**: P0（Phase 2）
- **适用场景**: 当你有一个模糊的活动想法（如"做个春节活动"），但没有明确的目标、指标和约束时。

## 场景目的
输入模糊的活动需求（如"做个春节活动"），输出结构化的活动设计目的说明书，明确：活动目标、目标用户、核心指标、时间节奏、资源约束。

## 输入模板

```markdown
## 活动设计目的梳理输入

### 需求来源
- 提出人/部门：_______
- 原始需求描述：_______（如："春节要搞个大活动，拉一波回流"）

### 背景信息
- 当前运营阶段：_______（如：公测3个月、DAU下降期）
- 近期数据表现：_______
- 竞品同期活动：_______

### 约束条件
- 活动上线时间：_______
- 活动持续时长：_______
- 预算/资源上限：_______
- 必须使用的系统/功能：_______
```

## 处理逻辑

1. **目标拆解**
   - 将模糊需求转化为SMART目标（具体、可衡量、可达成、相关、有时限）
   - 区分：核心目标（必须达成） vs 附加目标（尽量达成）

2. **目标用户分析**
   - 识别活动主要服务的用户群体
   - 分析目标用户的当前状态和需求痛点

3. **核心指标确定**
   - 确定1-2个北极星指标（活动成败的关键）
   - 确定3-5个辅助指标
   - 设定指标的基准值和目标值

4. **时间节奏规划**
   - 活动前预热期、活动中高潮期、活动后收尾期
   - 关键时间节点和对应运营动作

5. **资源与风险**
   - 所需资源清单（开发、美术、运营、推广）
   - 关键风险识别

6. **输出待澄清问题**
   - 至少5个需要确认的问题
