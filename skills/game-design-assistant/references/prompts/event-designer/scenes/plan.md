# Scene: EVT-plan - 活动方案生成

## 场景标识
- **ID**: EVT-plan
- **名称**: 活动方案生成
- **岗位**: 活动策划
- **优先级**: P0（Phase 2）
- **适用场景**: 当你已完成需求分析，需要输出包含规则详述、奖励清单、时间安排、资源配置的完整执行方案时。

## 场景目的
输入活动需求分析（EVT-design），生成完整、可直接执行的活动方案文档，包含：活动规则详述、奖励清单。

## 输入模板

```markdown
## 活动方案生成输入

### 前置文档
- [ ] 活动需求分析（EVT-design）

### 补充信息
- 活动预算上限：_______
- 开发工时上限：_______
- 美术工时上限：_______
- 必须上线的日期：_______
- 可接受的最晚上线日期：_______
```

## 处理逻辑

1. **方案框架搭建**
   - 基于需求分析搭建完整的方案结构
   - 确保所有需求都有对应的实现方案

2. **规则细化**
   - 将规则细化为程序可理解的逻辑
   - 包含所有边界条件和异常处理

3. **奖励配置**
   - 列出所有奖励的具体配置（ID、数量、概率）
   - 计算总成本，确保在预算内

4. **资源配置**
   - 明确开发、美术、配置所需的关键资源
   - 评估技术可行性与资源缺口
