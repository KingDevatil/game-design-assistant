# Scene: LVL-generate - 关卡概念生成

## 场景标识
- **ID**: LVL-generate
- **名称**: 关卡概念生成
- **岗位**: 关卡策划
- **优先级**: P0（Phase 2）

## 场景目的
输入关卡设计需求（主题、难度、玩法目标），生成完整的关卡概念方案，包括：关卡背景、布局思路、敌人配置、机关设计、奖励结构。


## 处理逻辑

1. **关卡定位**
   - 在整体关卡序列中的位置（教学/过渡/高潮/收尾）
   - 与前/后关卡的难度递进关系

2. **核心体验设计**
   - 确定本关的核心爽点/紧张点
   - 设计1-2个记忆点（让玩家印象深刻的时刻）

3. **布局规划**
   - 划分关卡阶段（开场 → 发展 → 高潮 → 收尾）
   - 每个阶段的场景元素和战斗节奏

4. **敌人波次设计**
   - 敌人类型、数量、出现时机
   - 波次间的节奏变化（紧张 → 放松 → 紧张）

5. **机关与陷阱**
   - 环境互动元素
   - 与叙事主题的呼应

6. **奖励与反馈**
   - 通关奖励
   - 隐藏要素（可选挑战）
