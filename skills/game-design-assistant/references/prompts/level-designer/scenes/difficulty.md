# Scene: LVL-difficulty - 关卡难度评估

## 场景标识
- **ID**: LVL-difficulty
- **名称**: 关卡难度评估
- **岗位**: 关卡策划
- **优先级**: P1（Phase 2）

## 场景目的
输入关卡配置（敌人属性、数量、机关、时间限制等），评估该关卡的综合难度，预测不同玩家群体的通关表现，输出难度调优建议。


## 处理逻辑

1. **难度因子拆解**
   - 敌人强度因子（HP × 攻击 × 数量）
   - 时间压力因子
   - 资源限制因子
   - 操作复杂度因子

2. **理论难度计算**
   - 计算综合难度评分
   - 与关卡序列中的其他关卡对比

3. **玩家分层预测**
   - 新手玩家：预计尝试次数、通关率
   - 普通玩家：预计尝试次数、通关率
   - 硬核玩家：预计尝试次数、通关率

4. **心流状态评估**
   - 评估挑战难度与玩家技能的匹配度
   - 识别可能导致焦虑（过难）或无聊（过易）的时段

5. **瓶颈识别**
   - 找出最容易导致玩家卡关的节点
   - 分析卡关原因（数值/操作/理解）

6. **调优建议**
   - 给出具体调整参数建议
